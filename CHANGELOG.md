# Changelog

## 0.1.1 - 2026-06-08

- Clear the product search input after adding simple products or variations to the cart.
- Close script-opened ticket print windows after the print dialog finishes.
- Add fallback cleanup for browsers that do not reliably fire the print lifecycle event.


All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-06

- Initial public open source release.
- Includes WooCommerce POS register, cash sessions, cash movements, payments, refunds, tickets, product search, customer lookup, audit logging, and admin configuration.
- Includes React source code and compiled frontend assets for WordPress installation.
